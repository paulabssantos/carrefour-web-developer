# TIPOS DE ERROS

## ECMAScript Error
  
    Ocorrem em tempo de execução

- Composto por:

    - Mensagem;
    - Nome;
    - Linha;
    - CallStack;

## DOMExcepction

    Erros relacionados ao Document Object Model (DOM)

- Erros referentes a estrutura do codigo que está sendo rodado em uma página da web.
